"Hello, World!" program

According to the Wikipedia "Hello, World!" program page

A "Hello, World!" program generally is a computer program that outputs or displays the message "Hello, World!".
Such a program is very simple in most programming languages, and is often used to illustrate the basic syntax of a programming language. It is often the first program written by people learning to code.
It can also be used as a sanity test to make sure that computer software intended to compile or run source code is correctly installed, and that the operator understands how to use it.

Web development: 3 components

The following description is a very simple example and is especially good for beginners to get to know the three most important components. Web development is a much larger field. Imagine building a house:

HTML
Building the basic structure of the house. So the foundation walls, maybe a basement, doors & windows and the roof.
CSS
Style the house. So you paint the walls in nice colors, the roof is going to be made red and the windows and doors are going to be painted with a light wood glaze.
JS
Bring functionality to the house. So you install a doorbell, lay the electricity and water pipes and install the all the appliances like telephone, TV or the kitchen oven.

Note that HTML is not a programming language

HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as

  • a stylesheet language like CSS
  • and scripting languages such as JS.
