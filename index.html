"Hello, World!" program

According to the Wikipedia article on the "Hello, World!" program.

A "Hello, World!" program generally is a computer program that outputs or displays the message "Hello, World!".
Such a program is very simple in most programming languages, and is often used to illustrate the basic syntax of a programming language. It is often the first program written by people learning to code.
It can also be used as a sanity test to make sure that computer software intended to compile or run source code is correctly installed, and that the operator understands how to use it.

Well, let's just say that this is my first "program" so I can get more familiar with web development and see if everything is working properly and as intended.

Web development

The following description is a very simple example and is especially good for beginners to get to know the three most important components of front-end web development. Note that web development is a much larger field. So, imagine building a house:

HTML
The basic structure of the house. So, constructing the foundation walls, perhaps a basement and installing doors, windows and the roof.
CSS
The look of the house. So, painting the walls in nice colors, turning the roof red and painting the windows and doors with a light wood finish.
JS
The functionality of the house. So, installing a doorbell, laying electricity and water pipes and installing all the appliances such as telephones, TVs or the kitchen stove.

HTML is not a programming language

HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as

  • a stylesheet language like CSS
    • also not a programming language
    • but indispensable for web development
  • and scripting languages such as JS.
    • a programming language
