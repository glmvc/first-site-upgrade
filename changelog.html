📋Changelog

⚙General

  • Space (Tab-Size) changed from 4 to 2 spaces (personal preference)
  • HTML, CSS and JavaScript comments explained in more detail
  • Code structures improved with comments and "separators"

HTML

  • HTML document language set to English (US): <html lang="en-US">
  • Meta description and author set within the head: <meta name="author" content="glmvc"> & <meta name="description" content="My Upgraded First Site">
  • Document title changed to "My Upgraded First Site" within the head: <title>My Upgraded First Site</title>
  • Favicon set within the head: <link rel="icon" href="favicon.ico" type="image/x-icon">
  • Script loading strategy changed and the script element is know within the head: <script src="scripts/main.js" defer></script>
  • Outgoing hyperlinks changed so that they open in a new tab instead of the current one: target="_blank" attribute
  • Title attribute added to hyperlinks and the image so hovering over them will open a descriptive tooltip: title="description" attribute
  • A video added to the main content (with fallback video): <video controls muted autoplay loop preload="auto" width="400">
  • Figure and figure caption added to the image and video: <figure>; <figcaption>
  • Quotations and citation added: <blockquote>; <q>; cite attribute; <cite>
  • Description list added to introduce HTML, CSS and JS: <dl>; <dt>; <dd>
  • Nested unordered list added
  • Abbreviations added to abbreviate HTML, CSS and JS: <abbr>
  • Character entity references added instead of HTML syntax characters: &amp; &apos; &quot; &lt; &gt;
  • Emojis added with character references: &#128025; &#128038;
  • Header (with navigation), main (with sections & sidebar) and footer structuring added: <header>; <nav>; <main>; <section>; <aside>; <footer>
  • Horizontal rules added to divide the sections: <hr>
  • Line breaks added to make paragraphs easier to read: <br>
  • SVG logo added inline to the header (with hyperlink): <svg></svg>
  • Social Media profiles with links added to the sidebar
  • A complex table about planet data added: <table></table>
  • Further sources attached to the ordered list
  • A soundtrack added to the footer (with fallback audio): <audio controls loop preload="auto">
  • Fake contact details added to the footer: <address>; href="tel:..." & href="mailto:..." attributes
  • Iframe for a map (from OpenStreetMap) added to the footer: <iframe>
  • Superscript added for the birthdate of the page within the footer: <sup>
  • Subscript added for the chemical formula within the footer: <sub>
  • A Japanese sentence added within the footer and language attribute set to Japanese: lang="ja" attribute
  • Date and time added for the last update and the birthdate of the page within the footer: <time datetime="YYYY-MM-ddThh:mm"></time>
  • Textual content adjusted to the changes
  • Code blocks added to the created source code page (code.html): (code, pre, var, kbd, samp)
  • Accessibility aspects changed and added where they were missing

CSS

  • External stylesheet, internal stylesheet and inline styling added to demonstrate it
  • Standard box model (content-box) changed to alternative box model for all elements in the document: box-sizing:border-box;
  • Global colors changed with variables
  • Type (tag/element), class and ID selectors now in use
  • Selector lists now in use
  • Descendant, child, adjacent and general sibling combinators now in use
  • Links styled with pseudo-class selectors
  • SVG icon attached to external links with an attribute selector (if link starts with "https://") and a special pseudo-element selector: a[href^="https://"]::after
  • Default margin & padding set to 0 ("reset stylesheet") with internal stylesheet using the universal selector: *{margin:0;padding:0}
  • Main heading (<h1>) set as inline box (outer display type) so that the click function really only affects the text and not the full width: display:inline;
  • Main heading (<h1>) wrapped in a div container to center it with an inner display type: display:flex; justify-content:center;
  • Subheading (<h2>) color set to whitesmoke with inline styling: style="color:whitesmoke;"
  • Calc function added to the width of the body: width:calc(80% - 50px);
  • Fallback width added for browsers that do not support the calc function: width:850px;
  • Image and main heading (<h1>) will now rotate a bit on hover: transform: rotate(0.005turn); transform: rotate(-0.001turn);
  • add background-image gradient
  • add rgba color value
  • Figure and figure caption styled
  • Table styled
  • Lists styled (ordered, unordered and description lists)
  • Button styled
  • The page is now full responsive (with media queries)

JavaScript

  • Figure caption also changing accordingly on clicking the image
  • Birthday Banner added to show on December 6 every year
  • Last edited date and time is now updating automatically
