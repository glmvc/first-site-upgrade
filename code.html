    <section class="web-dev">
      <h3>Web development: 3 components</h3>
      <p>
        The following description is a very simple example and is especially good for beginners to get to know the three most important components of front-end web development. Note that web development is a much larger field.<br>
        Imagine building a house:</p>
      <dl>
        <dt><abbr title="Hypertext Markup Language">HTML</abbr></dt>
        <dd>The basic structure of the house. So, constructing the foundation walls, perhaps a basement and installing doors, windows and the roof.</dd>
        <dt><abbr title="Cascading Style Sheets">CSS</abbr></dt>
        <dd>The look of the house. So, painting the walls in nice colors, turning the roof red and painting the windows and doors with a light wood finish.</dd>
        <dt><abbr title="JavaScript">JS</abbr></dt>
        <dd>The functionality of the house. So, installing a doorbell, laying electricity and water pipes and installing all the appliances such as telephones, TVs or the kitchen stove.</dd>
      </dl>
      <h4>HTML is not a programming language</h4>
      <p>
        <q cite="https://en.wikipedia.org/wiki/HTML"><strong><abbr title="Hypertext Markup Language">HTML</abbr></strong> is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as</q>
      </p>
      <ul>
        <li>a stylesheet language like <strong><abbr title="Cascading Style Sheets">CSS</abbr></strong>
          <ul>
            <li>also not a programming language</li>
          </ul>
        </li>
        <li>and scripting languages such as <strong><abbr title="JavaScript">JS</abbr></strong>.
          <ul>
            <li>a programming language</li>
          </ul>
        </li>
      </ul>
